Hash functions are **one-way transformations** of data into a fixed-length string.

## Properties
- Deterministic
- Irreversible
- Collision resistant
- Fast

## Common Hash Functions
- MD5 (128-bit)  
- SHA-1 (160-bit)  
- SHA-256 (256-bit)  
- HMAC (keyed hash for authentication)

### Applications
- Password storage
- File integrity verification
- Digital signatures
